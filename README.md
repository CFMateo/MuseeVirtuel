## Thème du projet

Nom du musée : Musée Guggenheim Bilbao

Description :
Ce site web présente une exposition virtuelle consacrée au Musée Guggenheim Bilbao, 
un musée d’art moderne et contemporain situé à Bilbao, en Espagne. Le site permet
de découvrir le musée, certaines œuvres importantes et une galerie d’images, comme
dans une visite virtuelle.
